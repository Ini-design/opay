# FicTechApp - OPay Clone Project

## Project Overview
This is a frontend-only OPay clone created as a **GenerativeAi student project**. It is not affiliated with or endorsed by OPay.

## Full HTML Source Code

## File Structure
- `index.html` - Main landing page (above)
- `style.css` - Styling for the main page
- `script.js` - JavaScript functionality
- `signup/index.html` - Sign up page
- `signup/style.css` - Sign up styling
- `signup/Script.js` - Sign up functionality
- `README.md` - This documentation file

## Key Features
1. **Header Navigation** - Logo, nav links, search, sign up button, dark mode toggle
2. **Hero Section** - Main banner with animated OPay card
3. **Features Section** - 4 main features: Payments, Transfers, Security, Savings
4. **Statistics** - Display of user stats and metrics
5. **Fee Calculator** - Interactive calculator for transfer fees
6. **Testimonials Carousel** - User reviews with navigation buttons
7. **Modals** - Sign up and transaction forms
8. **Live Chat Widget** - Support chat interface
9. **Footer** - Links and social media
10. **Dark Mode** - Theme toggle functionality

## Technologies Used
